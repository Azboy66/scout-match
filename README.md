# ⚽ Scout Match

Sélecteur de matchs à enjeux pour paris sportifs.

## Déploiement sur Vercel (gratuit)

### Étape 1 — Créer un compte GitHub
1. Va sur https://github.com et crée un compte gratuit
2. Clique sur **"New repository"**
3. Nomme-le `scout-match`
4. Clique sur **"Create repository"**

### Étape 2 — Uploader les fichiers
Sur la page de ton repo GitHub, clique sur **"uploading an existing file"**
et glisse-dépose tous les fichiers de ce dossier.

### Étape 3 — Déployer sur Vercel
1. Va sur https://vercel.com et connecte-toi avec GitHub
2. Clique sur **"Add New Project"**
3. Sélectionne ton repo `scout-match`
4. Clique sur **"Deploy"**
5. En 2 minutes, ton app est en ligne avec une URL du type :
   `https://scout-match-xxxx.vercel.app`

### Étape 4 — Installer sur ton téléphone
1. Ouvre l'URL sur ton téléphone (Chrome ou Safari)
2. **Android** : Menu ⋮ → "Ajouter à l'écran d'accueil"
3. **iPhone** : Bouton Partager → "Sur l'écran d'accueil"

## Utilisation
- Lance l'app et clique sur **"Connecter l'API"**
- Entre ta clé API-Football (gratuit sur api-football.com)
- Sélectionne tes championnats et critères
- Les matchs à enjeux s'affichent automatiquement !
